# User Workflow

## Worker Mode

1. Scan or enter the model number and select the circuit.
2. Confirm RPM and pressure; enter support-baseline readings.
3. Press **RUN**. The app shows an exact approved recipe only when one exists.
4. Install only the displayed recipe using its picture/reference point, then enter post-installation readings and press **VERIFY RESULT** in the controlled workflow.
5. PASS means every required measured braze/axis passed. FAIL means a measured requirement failed. NOT QUALIFIED means do not install a mass; contact an engineer.

## Engineer Mode

Engineers create and lock the 3D geometry and a revision-controlled recipe, attach measured evidence and approval, define the qualified envelope/absolute limit/tolerances, and may release only an empirical recipe. Model-derived candidates remain engineer-only.
