Rubber Mass Ring Selection Tool

Fast field calculator for discharge line vibration, ring mass selection, and placement records.

This version uses the full discharge line length entered by the worker. It does not ask for supports and does not shorten the span because the project is studying the line without support-based span reduction. The tool calculates the safety clearance internally. Workers only enter pipe length, pipe size, U-trap geometry, measured frequencies, and vibration readings.
